# hos

「[30日でできる！OS自作入門](https://www.amazon.co.jp/dp/B00IR1HYI0)」を参考に、OS自作を通じてコンピュータの低レイヤー（BIOS・アセンブリ・ブートローダ・x86アーキテクチャ）を手を動かしながら理解することを目的としたリポジトリです。

## やりたいこと

- x86アセンブリ・QEMUを使ったブートローダ・カーネルの自作
- BIOSの仕組みやブートシーケンスの理解
- C言語とアセンブリを組み合わせたOS開発の流れを習得

## ディレクトリ構成

| ディレクトリ | 内容 |
|---|---|
| `01day/` | 1日目：アセンブリでHello Worldをフロッピーイメージに書き込み、QEMUで起動 |
| `30daysOS/` | 書籍「30日でできる！OS自作入門」をLinux + GCCで実装した参考実装 |
| `playground/` | 実験・メモ用 |

## 開発環境

- OS: Ubuntu (on AWS)
- アセンブラ: GAS (GNU Assembler) / NASM
- エミュレータ: QEMU
- コンパイラ: GCC
